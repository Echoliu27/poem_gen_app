# poem_gen_app

Please download the model checkpoint at https://drive.google.com/file/d/1ouYjMeNglQ3SeI4go1T9OVLx9kubBtjc/view?usp=sharing
```
git clone repo
cd poem_gen_app
mkdir checkpoint
cd checkpoint
# download run1.zip
unzip run1.zip

pip install -r requirements.txt
```

To run app
```
python app.py
```
